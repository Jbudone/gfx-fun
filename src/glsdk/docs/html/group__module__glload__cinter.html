int LoadFunctions ( )

Loads all of the OpenGL function pointers for the current version and extensions.

This function will respect the core/compatibility flags that the context was created under (if the version is 3.2 or better and thus has those flags available). Similarly, if ARB_compatibility is defined for a 3.1 context, it will load the compatibility profile; otherwise, it will load the core. For 3.0 and below, it loads everything available.
